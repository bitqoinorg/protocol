# Contributing to Qonjoint Protocol

Thank you for your interest in contributing to the Qonjoint protocol specification.

## What This Repo Is

This repository contains the formal specification for the Qonjoint dual-key protocol.
It is intentionally separate from any implementation.

## How to Contribute

1. Fork the repository
2. Create a branch: `git checkout -b spec/your-change`
3. Edit or add files under `spec/`
4. Open a pull request with a clear description

## Rules

- Specs must be implementation-agnostic where possible
- Use precise language: "must", "should", "may" per RFC 2119
- No em-dashes anywhere
- All content in English

## Commit Format

```
docs(protocol): short description
```
